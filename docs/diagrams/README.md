# Diagrams

This folder contains all system diagrams, with Mermaid diagrams as the authoritative system views.

## Intended Content

- Architecture diagrams (Mermaid preferred)
- Sequence, flow, and component diagrams
- Deployment and infrastructure diagrams

## Structure

- Store each diagram as a separate file (preferably .md with Mermaid code blocks).
- Reference diagrams from architecture and ADR docs.
